Nuestros Servicios

Soluciones inteligentes para una mayor accesibilidad

Ofrecemos sistemas de iluminación adaptables diseñados para mejorar la autonomía y calidad de vida de las personas, integrando tecnología innovadora y soluciones accesibles.

👏

Sistema Principal

Control mediante aplausos

Nuestro sistema principal permite activar y controlar la iluminación utilizando aplausos, proporcionando una alternativa accesible y práctica para personas con discapacidades o movilidad reducida.

📱

Sistema Secundario

Aplicación móvil inteligente

Complementamos nuestra solución con una aplicación móvil que brinda una mejor adaptabilidad y control personalizado, permitiendo gestionar el sistema de iluminación de manera sencilla, eficiente y desde cualquier lugar.
